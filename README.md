# django-sspanel
用diango开发的全新的shadowsocks网络面板

Demo: [迷之屋](http://www.ehcozone.club/)

### 项目说明

* 前端CSS框架： BULMA(布玛)
* 通知功能： SweetAlert
* 后端： django 1.11
* 第三方包：qrcode,pillow
* so on...

### 支援ss版本

* shadowsocksr



### 使用方法

* 安装 lnmp
* 安装 git
* 安装 pyenv
* 安装 python-dev （sudo yum install python-devel mysql-devel ）
* 待编辑...
